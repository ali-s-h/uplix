## Installation with exe file

1 - download **[files](https://github.com/ali-s-h/uplix/raw/master/exe/setup.exe)**

2 - extract **file.zip**

2 - run **setup.exe**

### usage 

#### to upload file :

```console
%share% up <filename>
```
#### to download file :

```console
%share% dl <uploadkey>
``` 

## Installation with python

```console
pip install uplix
```

### usage  with python

#### to upload file :

```console
python3 -m uplix up <filename>
```
#### to download file :

```console
python3 -m uplix dl <uploadkey>
```
